# JSON examples

Este repositório contém um arquivo de dados (`dados.json`) e exemplos de payload para evitar o erro:

```
fields is required, at least one value must be present
```

## Arquivos

- `dados.json`: lista de itens com `codigo` e `descricao`
- `examples/ecm_form_request_fields_object.json`: exemplo onde `fields` é um objeto (map)
- `examples/ecm_form_request_fields_array.json`: exemplo onde `fields` é um array de campos

> Use o exemplo que corresponder ao formato exigido pelo endpoint do `ecm_form`.
